# Age of Mythology - Unciv Mod

Bring the legendary Age of Mythology experience to Unciv! This comprehensive mod adds four mythological civilizations, each with unique units, buildings, and abilities inspired by ancient legends.

## Civilizations

### Greeks - Led by Zeus
**Divine Favor**: Harness the power of the Olympian gods
- **Unique Ability**: +10% Strength for Military units during Golden Ages, +1 Culture from every Temple
- **Unique Units**: 
  - Hoplite (replaces Spearman): Enhanced anti-cavalry unit
  - Minotaur: Powerful melee unit that can move after attacking
  - Centaur: Fast ranged cavalry unit
  - Cyclops: Devastating siege unit
- **Unique Buildings**:
  - Temple of Zeus (replaces Temple): Provides production and culture bonuses
  - Parthenon: Grants significant culture and happiness

### Egyptians - Led by Ra
**Monuments of the Gods**: Build eternal monuments to the glory of Egypt
- **Unique Ability**: +20% Production when constructing wonders, +2 Gold from Desert tiles
- **Unique Units**:
  - Chariot Archer (replaces Chariot): Mobile ranged unit
  - Anubite: Powerful desert warrior that heals from kills
  - Sphinx: Mystical ranged unit with enhanced vision
  - Phoenix: Legendary flying unit
- **Unique Buildings**:
  - Temple of Ra (replaces Temple): Enhanced faith and gold generation
  - Pyramid: National wonder that boosts wonder production
  - Obelisk (replaces Monument): Enhanced culture from desert

### Norse - Led by Odin
**Berserker Rage**: Unleash the fury of the Nordic warriors
- **Unique Ability**: +15% Strength for Melee units when attacking, -15% maintenance for Military units
- **Unique Units**:
  - Berserk (replaces Longswordsman): Aggressive warrior with amphibious ability
  - Troll: Powerful unit that excels in cold climates
  - Valkyrie: Elite mounted unit that heals from kills
  - Frost Giant: Massive late-game unit
- **Unique Buildings**:
  - Temple of Odin (replaces Temple): Production bonus in tundra and snow
  - Longhouse (replaces Workshop): Enhanced production from forests
  - Mead Hall: Culture and happiness building

### Atlanteans - Led by Kronos
**Titans' Legacy**: Master advanced technology of the lost civilization
- **Unique Ability**: +2 Science from Coastal cities, +1 Production from Fishing Boats
- **Unique Units**:
  - Murmillo (replaces Swordsman): Defensive specialist
  - Automaton: Advanced mechanical warrior
  - Promethean: Self-replicating unit that can create improvements
  - Argus: Long-range observer with enhanced sight
- **Unique Buildings**:
  - Temple of Poseidon (replaces Temple): Science bonus in coastal cities
  - Sky Passage: Advanced science building for coastal cities
  - Mirror Tower (replaces Observatory): Enhanced science from mountains

## Special Features

### Mythological Technologies
- **Divine Intervention**: Enhances faith generation and temple benefits
- **Mythological Warfare**: Boosts mythological units
- **Sacred Architecture**: Accelerates wonder construction
- **Titan Technology**: Advanced late-game science boost

### World Wonders
- **Oracle**: Provides free Great Prophet and boosts faith
- **Colossus**: Enhances naval movement and provides gold

## Installation

1. Download the mod from the GitHub repository
2. Place the folder in your Unciv mods directory
3. Enable the mod in Unciv's mod manager
4. Start a new game and select one of the mythological civilizations

## Compatibility

This mod is designed to work with the base Unciv game. It adds new content without modifying existing civilizations, making it compatible with most other mods.
